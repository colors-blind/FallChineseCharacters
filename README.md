# 口字接物小游戏

一个简单的 HTML5 汉字小游戏：顶部会掉落“食物”和“非食物”汉字，玩家控制底部的“口”去接住掉落物。

- 接住**食物**：`口` 变大，分数 `+1`
- 接住**非食物**：`口` 变小，分数 `-1`（最低为 0）

## 项目文件

- `main.html`：页面结构与画布容器
- `styles.css`：页面与游戏界面样式
- `game.js`：核心玩法逻辑（掉落、碰撞、缩放、分数、输入控制）

## 本地运行

1. 进入项目目录：

2. 用浏览器直接打开 `main.html` 即可开始游戏。

## 操作方式

- 键盘：`←` / `→` 或 `A` / `D` 左右移动
- 鼠标/触控：在画布上拖动控制“口”位置

## 游戏规则

- 掉落文字分为两类：
  - 食物：`饭 面 果 鱼 菜 肉 粥 饼`
  - 非食物：`石 刀 火 雨 钉 尘 铁 沙`
- HUD 会实时显示：
  - 当前分数
  - `口` 的大小百分比
